Trabalho 1 - Relatório

O trabalho foi implementado utilizando o arquivo cisj.c disponibilizado na página do professor. Este arquivo possui a implementacao da função C(i,s) estudada em sala.

Foi utilizado uma matriz NxN onde a i-ésima linha corresponde ao vetor State[] do processo i. Para cada vetor State[], foi inicializado todos os processos no estado correto pois , como indicado pelo professor, assume-se que todos os processos iniciam no estado correto.

Em cada intervalo de teste, o testador(processo i) obtém os processos a serem testados utilizando a função C(i,s), com s variando de 1 a log N, então para cada processo j a ser testado, utilizando o vetor State[] do processo i é verificado se o processo i é o primeiro processo não falho em C(j,s), para o mesmo s anterior. Caso seja o primeiro processo valido, é atualizado o vetor State[] do processo i com as novidades do vetor State[] do processo j comparando elemento a elemento aquele que possui o maior valor,isto é, caso State_j[x] > State_i[x] para algum x, então State_j[x] possui uma novidade.
